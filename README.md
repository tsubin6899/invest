# 投資理財 Dashboard

這個資料夾是投資理財 Dashboard 的 GitHub Pages 靜態網站版本。

## 檔案

- `index.html`：Dashboard 首頁，可直接由 GitHub Pages 發布。
- `.nojekyll`：讓 GitHub Pages 以純靜態網站方式服務檔案。

## GitHub Pages 設定

在 GitHub repository 的 `Settings` -> `Pages`：

1. Source 選擇 `Deploy from a branch`
2. Branch 選擇 `main`
3. Folder 選擇 `/ (root)`
4. 儲存後等待 GitHub Pages 產生網址
